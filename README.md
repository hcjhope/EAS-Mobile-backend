## 开发背景

做EAS维护那么久了，不搞点事情完全不能混饭吃，所以开发这个项目。

## 技术栈

nodejs + express + mongodb + sqlserver + mongoose + sequelize + es6/7 + mocha + bluebird...

### 项目运行

```bash
git clone https://github.com/szy0syz/EAS-Mobile-backend.git

npm install
```
